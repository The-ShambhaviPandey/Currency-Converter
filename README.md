# Currency Converter

[Convert Your Money Now!](https://the-shambhavipandey.github.io/Currency-Converter/)

A simple yet elegant **JavaScript-based Currency Converter** web app that lets users convert one currency into another using real-time exchange rates.

---

## Features

- Converts between major global currencies
- Fetches **live exchange rates** from the [Frankfurter API](https://api.frankfurter.app)
- Clean and responsive UI built with HTML, CSS, and JavaScript
- Fast and lightweight — no frameworks required

---

## How It Works

The app fetches real-time exchange rates from this endpoint:
https://api.frankfurter.app/latest?from=USD&to=INR

It dynamically updates the converted value based on user input and selected currencies.

---

## Project Structure

| File           | Description                                                                        |
| -------------- | ---------------------------------------------------------------------------------- |
| **index.html** | The main HTML file — serves as the user interface of the converter                 |
| **style.css**  | Handles all the visual styling and layout of the web app                           |
| **script.js**  | Contains the logic for currency conversion, API integration, and DOM updates       |
| **code.js**    | Stores the list of available currency regions/codes used in the dropdown selection |

---

## Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript (ES6+)**
- **Frankfurter API** for real-time exchange data
